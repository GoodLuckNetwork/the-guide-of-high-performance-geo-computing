== NAME

lps2lts-mpi - Distributed state space generator.

== SYNOPSIS

mpirun _<nodespec>_ *lps2lts-mpi* [OPTION]... 'input.lps' ['output.fmt']

== DESCRIPTION

lps2lts-mpi performs a reachability analysis on a specification provided
in 'input.lps'.  Specifications are in _lps_ format and are commonly
generated by linkman:mcrl22lps[1].  'output.fmt' specifies the name of
the output archive.  The desired format is deduced from the filename
extension.
