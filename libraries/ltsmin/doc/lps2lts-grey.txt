== NAME

lps2lts-grey - Sequential state space generator

== SYNOPSIS

*lps2lts-grey* [OPTION]... 'input.lps' ['output.fmt']

== DESCRIPTION

lps2lts-grey generates a labelled transition system from a
specification provided in 'input.lps' by enumerative reachability
analysis.  Specifications are in _lps_ format and are commonly
generated by linkman:mcrl22lps[1].  'output.fmt' specifies the name
of the output archive.  The desired format is deduced from the
filename extension.
