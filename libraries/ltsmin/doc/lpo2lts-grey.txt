== NAME

lpo2lts-grey - Sequential state space generator for mCRL models.

== SYNOPSIS

*lpo2lts-grey* [OPTION]... 'file.tbf' ['outfile.fmt']

== DESCRIPTION

lpo2lts-grey generates a labelled transition system from a
specification provided in 'file.tbf' by enumerative reachability
analysis.  Specifications are in _tbf_ format and are commonly
generated by linkman:mcrl[1] or linkman:mcrl22mcrl[1].  'outfile.fmt'
specifies the name of the output archive.  The desired format is
deduced from the filename extension.
